# Vista Neotech - Digital Transformation Program

**Rebuilding WordPress → Next.js with Zero SEO Loss**

---

## 🎯 Strategic North Star

Vista Neotech Private Limited is migrating from WordPress to a modern Next.js platform while:
- **Preserving SEO authority** (zero ranking loss)
- **Elevating brand perception** (top-tier technology firm)
- **Building a lead capture engine** (conversion-first)
- **Scaling across multiple brands** (multi-tenant architecture)
- **Future-proofing** (performance, content velocity, analytics)

## 🛠️ Technology Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS + design tokens from DESIGN_SYSTEM.md
- **Language:** TypeScript
- **SEO:** Dynamic metadata from URL_MIGRATION_MAP, sitemap.xml, robots.txt, Organization JSON-LD
- **Database:** (Phase 5) Supabase or MS SQL for leads and content

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm (or pnpm/yarn)

### Install and run
```bash
npm install
npm run dev
```
Open [http://localhost:3000](http://localhost:3000). Build: `npm run build`.
